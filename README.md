# Tic Tac Toe Game

A modern, responsive Tic Tac Toe game built using HTML, CSS, and JavaScript.  
Playable on desktop and mobile browsers.

## Live Demo
https://swaraj808-dev.github.io/tic-tac-toe-js/

## Features
- Two-player gameplay
- Turn indicator
- Win and draw detection
- Restart game option
- Responsive design (mobile friendly)
- Modern glassmorphism UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Play
1. Player X starts first.
2. Players take turns marking a cell.
3. First to align 3 marks (row, column, diagonal) wins.
4. If all cells fill without a winner → draw.
5. Click Restart to play again.
